# QR_CODE_GENERATOR

A simple **QR Code Generator** built with **HTML, CSS, and vanilla JavaScript**.  
Generate QR codes from any text or URL and download the QR image instantly.

🌐 Live Preview:  
https://luigibarte4563.github.io/QR_CODE_GENERATOR/

---

## 🚀 Features

- 🔹 Generate QR codes for any text or URL  
- 📱 QR codes are scannable using phones or QR scanners  
- 💾 Download generated QR as PNG  
- 🧠 Clean and simple UI  
- ⚡ No backend required (Fully client-side)

---

## 🛠 How It Works

1. User enters text or URL in the input field  
2. JavaScript uses `qrcode.js` (via CDN) to generate the QR  
3. QR image is displayed instantly  
4. User clicks **Download QR** to save it as PNG

## 🧪 How to Use

1. Clone the repository:


git clone https://github.com/Luigibarte4563/QR_CODE_GENERATOR.git


2. Open `index.html` in your browser  

3. Enter any text or URL  

4. Click **Generate QR**  

5. Click **Download QR**
